Synopsis

webri: Serve ri documentation via the Web

Usage

webri [OPTIONS] … [RI_DIR]

OPTIONS: 

  -h, --help
     show help

  -o, --output OUTPUT_DIR
     ouput static files to OUTPUT_DIR

RI_DIR: 

  The directory in which to find the ri files generate by RDoc.
